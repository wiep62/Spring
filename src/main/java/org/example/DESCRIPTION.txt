1.5 Жизненный цикл ветка 1.5_Life_cycle_beans
1. Этап иниц-ии бинов:
 - Поиск всех бинов которые были объявлены через аннотации, конфигурации  и создают на базе их специальные объекты (BeanDifinition) для каждого бина
 BeanDifinition - объект, который содержит информацию о том как настраивать и создавать конкретный бин
 - созданые BeanDifinition регистрируются в контейнере (они будут исп-ся для создания и регулирования жизн-го цикла БИНА)
 - с помощью рефлексии созд-ся экз-ры бинов, а также передаются нужные зависимости
 - контейнер уст-ет нужные свойства для настройки бина
 - контейнер проверяет, что все свойства были установленны правильно
 - вызов методов, которые помечены как ПОСТ - ИНИЦИАЛИЗАЦИОННЫЕ
 - создание proxy-объектов (могут обогащать БИН дополнительной логикой)
